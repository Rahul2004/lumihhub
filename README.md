# ⚡ Lumina Hub - Creative All-in-One Workspace

**Lumina Hub** is a premium, open-source collection of creative and productivity tools designed for speed and privacy.

## 🚀 Key Features:
- **Image Editor**: Add filters, adjust brightness, contrast, and Download in multiple formats.
- **Magic Eraser (AI)**: Automatically scan and remove backgrounds from any image using locally-run AI (BodyPix).
- **QR Generator**: Instantly convert any link or text into a high-definition download-ready QR code.
- **File Transformers**: Convert between JSON and CSV formats instantly.
- **PDF Toolkit**: Merge PDFs, extract images from PDFs (as ZIPs), and convert PDFs to Word or Text.
- **Image Compressor**: Optimize and shrink image file sizes without sacrificing quality.

## 🎨 Built with:
- **HTML5 & Vanilla CSS**: Ultra-fast structure and modern design system.
- **JavaScript (ES6+)**: High-performance, client-side logic—no uploads required!
- **TensorFlow.js & BodyPix**: Powerful, privacy-first AI background removal.
- **PDF.js, jsPDF, & JSZip**: Advanced file manipulation directly in the browser.

## 📝 How to use:
Simply host this repository on any static site hosting service (like **GitHub Pages**) and you're good to go!
